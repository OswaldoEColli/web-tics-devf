Frontend

El frontend es la cara visible de una aplicación web. Aquí se desarrolla la interfaz con la que interactúan los usuarios.

  • Lenguajes: HTML, CSS, JavaScript
  • Frameworks: React, Angular
  • Herramientas: Bootstrap, Tailwind

Backend

El backend gestiona la lógica del negocio, las bases de datos y la comunicación con el frontend mediante APIs.

  • Lenguajes: Python, Java, Node.js
  • Frameworks: Django, Laravel
  • Bases de datos: MySQL, MongoDB
